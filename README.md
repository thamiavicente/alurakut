## :arrow_forward: How to run
<p>1. Clone the projet </p>

```
git clone https://github.com/thamiavicente/alurakut.git
```
<p>2. Install all dependencies </p>

```
yarn install
```
<p>3. Start project </p>

```
yarn dev
```
<p>4. Access the project's localhost. The default port is 3000</p>
